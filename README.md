RoomPool

RoomPool is a room sharing and roommate matching platform.

Features
- User authentication
- Room listing
- Room search and filtering
- Roommate matching
- Booking system
- Rent splitting
- In-app chat

Tech Stack
Frontend: React  
Backend: Python (FastAPI)  
Database: PostgreSQL  
Architecture: Microservices

Team Workflow
- main → production
- develop → integration
- feature branches → development
